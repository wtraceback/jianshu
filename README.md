# React 模仿简书的部分功能

react + redux + react-router + ES6 + immutable + axios

# 说明

本项目主要用于理解 react 和 redux 的编译方式，以及熟悉 react + redux 配合构建网站的方式

# 项目截图
![简书首页](screenshot/index_with_jianshu.png)

# 项目功能
- 首页
- 内容详细页
- 登录注册页
- 登录鉴权

# 项目运行
```bash
# 克隆项目
git clone https://github.com/wtraceback/react-jianshu.git
# 切换目录
cd react-jianshu
# 安装依赖
npm install
# 运行项目
npm run start
```
